# 线性树（链式树）

## 一、核心定义

线性树，又称**链式树、链状树、单链树**，是树的一种特殊形态，核心特征是所有节点连成一条直线，无任何分叉，本质是「链表形状的树」。

核心定位：树的简化形态，可理解为“没有分支的树”，兼具树的无环特性和链表的线性结构。

## 二、结构特征

满足以下所有条件，即为线性树：

- 结构上：只有一条主链，**无任何分叉**，所有节点依次串联，没有兄弟节点。

- 度数特征（节点度数 = 直接相邻的节点数）：

    - 根节点（通常为链首节点，如节点1）：度数 = 1（仅连接下一个节点）；

    - 中间节点（链中所有非首尾节点）：度数 = 2（连接前一个和后一个节点）；

    - 叶子节点（链尾节点）：度数 = 1（仅连接前一个节点）。

- 路径特征：任意两个节点之间的路径**唯一**，且路径就是整条主链。

- 数量关系：节点总数 = 整条链的路径长度 \+ 1（如5个节点的线性树，路径长度为4）。

## 三、直观结构演示

1\. 简化链式表示

```
1 → 2 → 3 → 4 → 5
```

2\. 标准树形画法

```
1
|
2
|
3
|
4
|
5
```

## 四、与普通树的区别

| 对比维度   | 线性树（链式树）    | 普通树            |
|--------|-------------|----------------|
| 结构形态   | 一条直线，无分叉    | 有分叉，存在分支节点     |
| 分叉情况   | 无任何分叉，无兄弟节点 | 有分叉，存在兄弟节点     |
| 叶子节点数量 | 仅 1 个（链尾节点） | ≥ 2 个          |
| 中间节点度数 | 均为 2        | ≥ 2（分支节点度数 ≥3） |

## 五、一句话总结

线性树 = 拉直成一条直线的树，无任何分叉、无兄弟节点，节点依次串联，兼具树的无环性和链表的线性性。

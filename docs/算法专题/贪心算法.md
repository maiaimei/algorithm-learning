# 贪心算法

## 1. 什么是贪心算法？

**核心思想**：

在对问题求解时，总是做出在当前看来是最好的选择。也就是说，不从整体最优上加以考虑，它所做出的仅仅是在某种意义上的**局部最优解**。

**特点：**

1. **贪心选择性质**：一个全局最优解可以通过一系列局部最优的选择来达到。这是贪心算法可行的第一个基本要素。
2. **无后效性**：一旦做出了选择，就无法改变。当前的选择只依赖于当前的状态，不依赖于未来的状态或之前的选择对未来的影响。
3. **自顶向下**：通常是以迭代的方式做出当前的贪心选择，然后解决剩下的子问题。

**适用场景：**

- 贪心能得到**全局最优**：活动选择（选结束时间最早的活动，以便留出更多时间给后续活动）、霍夫曼编码、最小生成树、最短路径、零钱兑换（假设硬币面额是 1, 5, 10, 20，要凑出 37 元，贪心策略是先拿最大的
  20，再拿 10……）。
- 贪心只能得到**近似解**：背包问题、旅行商问题。

## 2. 贪心经典题型体系

### 2.1 区间调度系列

**问题**：给定一组**时间区间**（每个区间有开始时间 `start` 和结束时间 `end`），要求选出**最多数量**的**互不重叠**区间。

**策略**：按**结束时间升序**排序，**优先选择结束时间最早的区间**。

**特点：**

1. 有一组区间（开始时间 + 结束时间）
2. 区间不能重叠（重叠就冲突）
3. 目标：选最多不重叠的区间

**典型题**：

- 最多能参加多少场会议
- 最多安排多少个活动
- 移除最少区间，让剩下的不重叠
- 最多不重叠子区间数量
- 会场安排、教室使用、任务调度
- 射气球问题

对应LeetCode题目编号（高频经典）：

- \435. 无重叠区间（核心基础题，对应“移除最少区间”，中等）
- \452. 用最少数量的箭引爆气球（射气球问题，中等）
- \56. 合并区间（区间调度延伸题，中等）
- \253. 会议室 II（会场安排问题，中等）
- \1024. 视频拼接（区间覆盖类，贪心延伸，中等）

**解题步骤（万能模板）：**

1. 把所有区间按**结束时间从小到大排序**
2. 初始化
    1. 记录上一个选中区间的结束时间
    2. 记录选中的数量
3. 遍历所有区间
    1. 如果当前区间**开始时间 ≥ 上一个结束时间** → 选中
    2. 更新结束时间，数量 + 1
4. 返回结果

**总结：**

1. 区间调度 = 选最多不重叠区间
2. 识别：区间 + 不重叠 + 最多数量
3. 贪心策略：按**结束时间升序排序**
4. 代码模板：排序 → 选第一个 → 遍历判断不重叠
5. 时间复杂度：O(n log n)（排序耗时）

**一句话记忆：**区间调度 = 按结束时间排序 + 选不重叠 = 贪心最优

### 2.2 跳跃游戏系列（高频）

#### （1）跳跃游戏 I（能否到达终点）

- 题目：数组中每一项表示最大跳跃步数，问能否跳到最后。
- **贪心策略**：维护当前能到达的**最远距离**。
- 遍历每个位置，如果当前位置 > 最远距离 → 不可达。

对应LeetCode题目编号：

- \55. 跳跃游戏（核心题，直接考察贪心策略，中等）

#### （2）跳跃游戏 II（最小跳跃次数）

- 题目：求跳到终点的**最少步数**。
- **贪心策略**：
    - 在当前可跳范围内，选能跳得**最远**的位置。
    - 到达边界时必须跳一步，步数+1。

对应LeetCode题目编号：

- \45. 跳跃游戏 II（高频进阶题，中等）

### 2.3 分发糖果/分饼干（分配类贪心）

#### （1）分发饼干

- 题目：给孩子分饼干，孩子有胃口，饼干有大小，求最多满足几个孩子。
- **策略**：
    - 孩子胃口、饼干大小都**从小到大排序**。
    - 小饼干先喂饱小胃口。

对应LeetCode题目编号：

- \455. 分发饼干（核心基础题，简单）

#### （2）分发糖果

- 题目：每个孩子至少 1 颗糖，评分更高的孩子必须比旁边多。
- **策略**：
    - 左到右扫一遍：右边 > 左边 → 右边 = 左边+1
    - 右到左扫一遍：左边 > 右边 → 左边 = max(左边, 右边+1)

对应LeetCode题目编号：

- \135. 分发糖果（高频进阶题，困难）

### 2.4 股票买卖系列（简单贪心）

适合可无限次交易的情况：

**策略**：只要后一天 > 前一天，就赚差价。

每天最优 → 全局最优。

对应LeetCode题目编号（贪心可解）：

- \122. 买卖股票的最佳时机 II（核心题，无限次交易，简单）
- \714. 买卖股票的最佳时机含手续费（延伸题，无限次交易+手续费，中等）

### 2.5 加油站问题

- 题目：环形路线，每个加油站有油，到下一站消耗油，问能否绕一圈。
- **贪心策略**：
    - 总剩余油量 < 0 → 不可能。
    - 一旦当前累计为负，说明起点不可能在前面，更新起点为下一站。

对应LeetCode题目编号：

- \134. 加油站（核心经典题，中等）

### 2.6 最大匹配/字典序类

#### 移掉 K 位数字使结果最小

- 策略：维护单调递增栈，尽可能删除前面大的数字。

对应LeetCode题目编号：

- \402. 移掉 K 位数字（核心题，中等）

#### 拼接数字最小/最大

- 例：`[10,2]` 可拼成`102` 或 `210`，选最小。
- 策略：按 `a+b < b+a` 排序。

对应LeetCode题目编号：

- \179. 最大数（拼接最大数，贪心排序核心，中等）

### 2.7 霍夫曼编码（贪心经典）

- 每次选**两个最小权值**节点合并。
- 用于数据压缩，构造带权路径长度最短的二叉树。

对应LeetCode题目编号（霍夫曼思想应用）：

- \863. 二叉树中所有距离为 K 的结点（霍夫曼编码思想求解，中等）
- \1167. 连接棒材的最低费用（霍夫曼编码核心应用，中等）

### 2.8 图论中的贪心算法

#### 最小生成树 MST

- **Kruskal**：边按权值从小到大，不形成环就选。
- **Prim**：从点出发，每次选连接当前集合的最小边。

对应LeetCode题目编号：

- \1584. 连接所有点的最小费用（Prim/Kruskal均可解，中等）
- \684. 冗余连接（Kruskal思想延伸，判环，中等）

#### 最短路径

- **Dijkstra**：每次选**距离源点最近**的点松弛。

对应LeetCode题目编号：

- \743. 网络延迟时间（Dijkstra核心题，中等）
- \1631. 最小体力消耗路径（Dijkstra变种题，中等）

## 3. 贪心算法通用判断技巧

- 能分解为**一步一步选择**
- 每一步选择**不影响后续**
- 局部最优能推出**全局最优**
- 通常需要**排序**后贪心

> 不是所有问题都能用贪心，例如普通背包、硬币非规范面额时会错。

## 4. 贪心 vs 动态规划

- **贪心**：只选一条路，不回头，快。
- **DP**：遍历所有可能，存最优子结构，准。
- 贪心正确的前提：**具有贪心选择性质 + 最优子结构**。

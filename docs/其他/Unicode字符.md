# 数学常用 Unicode 字符大全

## 一、基本运算符

```
+ 加     - 减    × 乘   ÷ 除
± 加减   ∓ 减加  ∙ 点乘
```

## 二、比较符号

```
= 等于   ≠ 不等于   ≡ 恒等于
≈ 约等于 ≃ 渐近等于 ≉ 不约等于
< 小于   > 大于     ≤ 小于等于
≥ 大于等于 ≪ 远小于 ≫ 远大于
```

## 三、集合符号

```
∈ 属于   ∉ 不属于   ∋ 包含
⊂ 子集   ⊃ 超集    ⊆ 子集或等于
⊇ 超集或等于 ∪ 并集 ∩ 交集
∅ 空集   ∀ 任意存在 ∃ 存在
∄ 不存在
```

## 四、希腊字母（常用于变量）

**大写**：

```
Α Β Γ Δ Ε Ζ Η Θ Ι Κ Λ Μ
Ν Ξ Ο Π Ρ Σ Τ Υ Φ Χ Ψ Ω
```

**小写**：

```
α β γ δ ε ζ η θ ι κ λ μ ν
ξ ο π ρ σ ς τ υ φ χ ψ ω
```

常用：θ, π, φ, σ, μ, λ, ω, α, β, γ, ε

## 五、特殊常数

```
π 圆周率 ≈ 3.1416
ℯ 自然对数底 ≈ 2.7183
φ 黄金比例 ≈ 1.618
i 虚数单位
```

## 六、上标数字

```
⁰ ¹ ² ³ ⁴ ⁵ ⁶ ⁷ ⁸ ⁹ ⁺ ⁻ ⁼ ⁽ ⁾
```

示例：x² + y² = z²

## 七、下标数字

```
₀ ₁ ₂ ₃ ₄ ₅ ₆ ₇ ₈ ₉ ₊ ₋ ₌ ₍ ₎
```

示例：A₁, B₂, x₀

## 八、积分和微积分

```
∫ 积分   ∮ 闭合积分
∂ 偏微分 ∇ 纳布拉算子
∆ 拉普拉斯算子
′ 撇（导数）  ″ 两撇
```

## 九、箭头符号

```
→ 向右箭头  ← 向左箭头
↔ 双向箭头  ↑ 向上
↓ 向下      ⇒ 蕴含
⇔ 等价
```

## 十、几何符号

```
∠ 角       ∟ 直角
∥ 平行     ∦ 不平行
⊥ 垂直     △ 三角形
□ 正方形   ○ 圆
```

## 十一、逻辑符号

```
¬ 非       ∧ 与
∨ 或       → 蕴含
↔ 等价     ⊕ 异或
⊤ 真       ⊥ 假
```

## 十二、其他数学符号

```
√ 平方根    ∛ 立方根
∞ 无穷大    ℵ 阿列夫数
% 百分号    ‰ 千分号
∑ 求和      ∏ 求积
∫ 积分     ∮ 环路积分
```

## 十三、分数

```
¼ ½ ¾ ⅓ ⅔ ⅕ ⅖ ⅗ ⅘
```

## 实用技巧

### 1. 键盘输入（Windows）

- Alt + 小键盘数字
- 示例：Alt+0177 = ±，Alt+247 = ÷

### 2. 输入法

- 拼音输入法：输入`pingfang`找²
- 搜狗：输入`dui`找✓
- 五笔：部分符号有编码

### 3. 在线工具

- Unicode字符表
- 数学符号生成器
- 复制粘贴常用符号

## 常用组合示例

```
x² + y² = z²       勾股定理
∑_{i=1}^n i = n(n+1)/2  求和公式
∫₀¹ x² dx = ⅓       积分
a ∈ ℝ, b ∈ ℂ        实数、复数
θ = 45°, α = π/4    角度
∂f/∂x = ∂f/∂y      偏导
∀ε>0, ∃δ>0         极限定义
```

## 记忆技巧

1. **希腊字母**：记住αβγδε, θπφσω
2. **运算符**：≠≤≥≈∫∑
3. **上下标**：上标在字母右上，下标在右下
4. **特殊符号**：常用如∈∞π√≈±

这些符号可以在大部分现代软件和网页中正常显示，包括Word、Excel、浏览器、编程IDE等。

# ASCII 字符编码表

## ASCII 表概述

ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是一种基于拉丁字母的字符编码标准，主要用于现代英语。它使用7位二进制数（0-127）表示128个字符。

## ASCII 表的重要规律

### 1. 控制字符 (0-31, 127)

- 0-31 和 127 是**控制字符**，不可打印
- 包括通信控制、格式控制、设备控制等
- 常用控制字符：
    - **9 (HT)**：水平制表符（Tab键）
    - **10 (LF)**：换行符（Unix/Linux/macOS换行）
    - **13 (CR)**：回车符（Windows换行是CR+LF）
    - **27 (ESC)**：转义符

### 2. 可打印字符 (32-126)

- **32 (SP)**：空格
- **48-57**：数字 0-9
- **65-90**：大写字母 A-Z
- **97-122**：小写字母 a-z
- 其余为标点符号和特殊符号

### 3. 大小写字母转换

- 大小写字母相差 32（小写 ASCII 码 = 大写 ASCII 码 + 32）
- 大写 A(65) → 小写 a(97)，差值为 32
- 快速转换：`大写字母 | 0x20 = 小写字母`，`小写字母 & 0xDF = 大写字母`

### 4. 数字字符

- 数字字符 '0'-'9' 的 ASCII 码是 48-57
- 数字字符转数字值：`数字字符 - '0'`或 `数字字符 - 48`
- 例如：'5'(53) - '0'(48) = 5

## 扩展 ASCII (128-255)

标准 ASCII 只定义了 0-127，128-255 是扩展 ASCII，不同系统和语言有不同的定义：

- **ISO-8859-1**（Latin-1）：西欧语言
- **Windows-1252**：Windows 西欧语言
- **GBK/GB2312**：中文扩展
- 现代通常使用 **UTF-8** 编码

## ASCII 在编程中的应用

### C/C++ 示例

```c
char ch = 'A';
printf("%d", ch);  // 输出 65
printf("%c", 97);  // 输出 'a'
```

### Python 示例

```python
ord('A')  # 返回 65
chr(97)   # 返回 'a'
```

### JavaScript 示例

```java
'A'.charCodeAt(0);  // 返回 65
String.fromCharCode(97);  // 返回 'a'
```
